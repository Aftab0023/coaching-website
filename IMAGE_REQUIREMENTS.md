# Website Image Requirements

Add the images below when real academy photos are available. Place all files in `public/images/` and keep the same filenames so they can be used directly in the React components.

| File | Where to use | What image should show | Suggested size |
| --- | --- | --- | --- |
| `academy-classroom.jpg` | Hero section | Bright, real classroom photo with students learning and a teacher visible | 1600 x 1000 |
| `faculty-teaching.jpg` | About section | Teacher explaining on a board or helping a student one-to-one | 1200 x 900 |
| `students-studying.jpg` | Courses/results sections | Focused students writing tests or studying in class | 1200 x 900 |
| `academy-front.jpg` | Contact section | Outside/front photo of the academy building or entrance | 1200 x 800 |
| `topper-wall.jpg` | Results section | Real result board, certificates, toppers wall, or award photo | 1200 x 800 |

Image notes:

- Use real academy photos instead of stock photos wherever possible.
- Keep faces clear only if you have permission from students/parents.
- Compress images before adding them so each file is ideally under 350 KB.
- Avoid dark, blurry, heavily filtered, or very cropped photos.
